【原理】

（1）For循环

for(init counter;test counter;increment counter){

​ code to executed;

}

参数

init counter：初始化循环计数器的值

test counter：评估每个循环。如果值为TRUE，继续循环。如果它的值为FALSE，循环结束

invrement counter：增加循环计数器的值

（2）Foreach

Foreach遍历数组中每个元素并循环代码块，每进行一次循环，当前数组元素的值就会被赋值给$value变量，并且数组指针会逐一地移动，直到到达最后一个数组

foreach($array as $value){

​ code to executed;

}
